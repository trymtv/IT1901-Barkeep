### To run the project:

*We need Spring boot to run when running GUI-tests*

1. `mvn -pl core install`
2. `mvn -pl core spring-boot:start`

3. `mvn install`

**To show GUI:**

4. `mvn -pl fxgui javafx:run`

**Make sure spring-boot is running (see point 2)**
